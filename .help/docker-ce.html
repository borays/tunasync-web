Docker-CE 镜像使用帮助

说明

本镜像只提供 Debian/Ubuntu/Fedora/CentOS/RHEL 的 docker 软件包，非 dockerhub

CentOS/RHEL/Fedora 用户

如果你之前安装过 docker，请先删掉

yum remove docker docker-common docker-selinux docker-engine

安装一些依赖

yum install -y yum-utils device-mapper-persistent-data lvm2

下载对应版本repo文件, 放入/etc/yum.repos.d/

  • CentOS7/Redhat7
  • Fedora

设置镜像源URL，例如：镜像URL地址为192.168.1.205时

方法一：
echo "192.168.1.205 URL" >> /etc/hosts
方法二：
sed -i s#URL#192.168.1.205# /etc/yum.repos.d/docker-ce.repo

运行以下命令更新软件包缓存

yum clean all
yum makecache

最后安装docker-ce

yum install docker-ce

Debian 用户

如果你之前安装过 docker，请先删掉

apt-get remove docker docker-engine docker.io

安装一些依赖

apt-get install apt-transport-https ca-certificates curl gnupg2 software-properties-common

信任 Docker 的 GPG 公钥:请将以下URL替换成镜像源地址或增加hosts

curl -fsSL http://URL/linux/debian/gpg | sudo apt-key add -

对于 amd64 架构的计算机，添加软件仓库:

sudo add-apt-repository \
   "deb [arch=amd64] http://URL/docker-ce/linux/debian \
   $(lsb_release -cs) \
   stable"

如果你是树莓派或其它ARM架构计算机，请运行:

echo "deb [arch=armhf] http://URL/docker-ce/linux/debian \
     $(lsb_release -cs) stable" | \
    sudo tee /etc/apt/sources.list.d/docker.list

最后安装docker-ce

apt-get update
apt-get install docker-ce
