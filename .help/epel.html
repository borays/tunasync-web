EPEL 镜像使用帮助

说明

EPEL (Extra Packages for Enterprise Linux) 是由 Fedora Special Interest Group 为企业 Linux 创建、维护和管理的一个高质量附加包集合，适用于但不仅限于 Red Hat Enterprise Linux (RHEL), CentOS, Scientific Linux (SL), Oracle Linux (OL)等。

使用说明

首先确认 /etc/yum.repos.d/ 目录下没有epel的源

下载对应版本repo文件, 放入/etc/yum.repos.d/

  • CentOS7/Redhat7
  • CentOS6/Redhat6

设置镜像源URL，例如：镜像URL地址为192.168.1.205时

方法一：
echo "192.168.1.205 URL" >> /etc/hosts
方法二：
sed -i s#URL#192.168.1.205# /etc/yum.repos.d/CentOS*-epel.repo

运行以下命令更新软件包缓存

yum clean all
yum makecache
